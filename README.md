# MoveUp Fitness

Este é o repositório da landing page MoveUp Fitness, construída com Vite, React, TypeScript, Tailwind CSS e shadcn/ui.

## Requisitos
- Node.js 18+
- npm

## Desenvolvimento
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
3. Acesse o projeto em `http://localhost:8080` (ou a porta escolhida automaticamente).

## Build e Preview
- Build: `npm run build`
- Preview do build: `npm run preview`

## Estrutura
- `src/`: código fonte (componentes, páginas, estilos)
- `public/`: assets estáticos (favicon, imagens)
- `vite.config.ts`: configuração do Vite
